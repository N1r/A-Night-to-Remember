## 🟢 第一阶段：生存指南 (Survival)

Neovim 最让人崩溃的是“动不了”。记住，它有**模式**之分：

* **Normal 模式**（按 `Esc` 进入）：这是你的“指挥部”。所有的快捷键（移动、删除、复制）都在这里。
* **Insert 模式**（按 `i` 进入）：这是你的“码字房”。只有这里才能像普通编辑器一样打字。
* **Command 模式**（按 `:` 进入）：这是你的“控制台”。输入 `:w` 保存，`:q` 退出。

### **博士级必背移动键 (Normal 模式)**

不要用方向键！手不要离开主键盘区：

* **`h` `j` `k` `l**` $\rightarrow$ **左 下 上 右**
* **`w`**：跳到下一个单词开头（Word）
* **`b`**：跳回上一个单词开头（Back）
* **`u`**：撤销（Undo，这就是你的后悔药）

## 🟡 第二阶段：现代人的捷径 (LazyVim)

别去从零开始写 `init.lua` 配置，那太费时间了。作为 **D（博士）**，我们要学会站在巨人的肩膀上。

直接安装 **LazyVim**（目前最火、最现代的配置框架）：

1. 备份你现有的配置（如果有）。
2. 克隆它的模板到你的配置目录：
```bash
git clone https://github.com/LazyVim/starter ~/.config/nvim

```


3. 启动 `nvim`，它会自动下载所有插件。

**为什么推荐 LazyVim？**

* 它内置了 **Lazygit** 集成（直接在 Nvim 里按 `<leader>gg` 就能唤起你熟悉的 Lazygit）。
* 自带 **Telescope**（按 `<leader>ff` 全局搜索文件，像 B 站搜索一样快）。
* 代码补全、语法检查全是现成的。

## 🟠 第三阶段：进阶操作 (The "D" Power)

当你熟悉了基本移动，试试这些组合拳：

* **`ciw`** (Change Inner Word)：直接删掉当前单词并进入输入模式。比鼠标双击再按 Backspace 快 10 倍。
* **`y`** (Yank) 是复制，**`p`** (Paste) 是粘贴。
* **`v`** (Visual) 是选区模式，配合 `j/k` 可以快速选中一大块代码。

### **给你的第一个任务：**

如果你已经装好了 Neovim，试着打开一个文件，练习：

1. 按 `i` 随便写几行字。
2. 按 `Esc` 回到 Normal 模式。
3. 用 `k` 和 `j` 上下移动。
4. 输入 `:w` 保存。
*既然你已经有了 **D（博士）** 的逻辑思维，Neovim 的快捷键就不再是死记硬背，而是一套**“动词 + 名词”**的语法。

在 Neovim 里，你可以像拼句子一样操作代码。以下是公认最顺手、最能提升效率的快捷键：

---

### **1. 基础移动：手不离位 (Normal 模式)**

* **`gg`**：直达文件顶部（第一行）。
* **`G`**：直达文件底部（最后一行）。
* **`0`**（零）：跳到行首。
* **`$`**：跳到行尾。
* **`f` + `字符**`：行内搜索。比如想跳到当前行的 `(`，按 `f(` 即可，比狂按 `l` 快得多。

---

### **2. 核心“骚操作”：动词 + 范围**

这是 Neovim 的精髓。掌握这三个动词，你就是代码魔术师：

* **`d` (Delete)**：删除
* **`c` (Change)**：删除并进入编辑模式
* **`y` (Yank)**：复制

**配合范围（Inner/Around）：**

* **`ciw`**：修改当前单词（Change Inner Word）。**最常用！**
* **`di"`**：删除引号内的内容（Delete Inner "）。处理字符串神器。
* **`ca(`**：修改括号及括号内的内容（Change Around (）。
* **`dG`**：从当前位置一直删到文件末尾。

---

### **3. 高效编辑：懒人必备**

* **`o`** (小写)：在当前行**下方**插入新行并进入编辑模式。
* **`O`** (大写)：在当前行**上方**插入新行。
* **`x`**：删除光标处的单个字符。
* **`.`** (点号)：**重复上一次操作**。这是 Neovim 最神的功能，如果你刚删了一个词，移动到下一个词按 `.` 就能接着删。

---

### **4. 屏幕与窗口管理**

如果你在写论文（D）时需要对照参考资料：

* **`zz`**：将当前行居中显示（在看长难段落时非常有安全感）。
* **`<C-w>v`**：左右分屏（Ctrl + w 然后按 v）。
* **`<C-w>h/j/k/l`**：在不同窗口间切换。

---

### **5. 现代插件增强（如果你用了 LazyVim）**

* **`<leader>ff`**：全文件名搜索（Find Files）。
* **`<leader>fw`**：全局文字搜索（Find Word），就像在整个硬盘里搜关键词。
* **`<leader>e`**：打开左侧文件树，像 VS Code 那样预览目录。
* **`<leader>gg`**：**直接在 Neovim 里唤出你熟悉的 Lazygit**。

---

### **💡 博士级的“刻意练习”建议**

你可以把这些快捷键融入你的 **A-E 计划**：

1. **A (A股)**：操作要快，止损要狠。练好 `u` (撤销) 和 `Ctrl-r` (重做)。
2. **C (Core)**：先练熟 `ciw` 和 `f`，这两个能解决 80% 的日常定位和修改需求。
3. **E (Exercise)**：每天尝试**不用鼠标**写一段代码或笔记。
